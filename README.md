
## Assortment of useful tools

### LLMs
* **Claude Code** — Examples of skills, sub-agents, and hooks

### Development
* **plantuml.md** — Draw graphs and diagrams; includes a C4 diagram prompt

### Terminal Utilities
* **tracker/** — Terminal task tracker with time logging. Curses UI for starting, pausing, and completing tasks; tracks cumulative time across sessions. Data stored as editable Markdown tables.
* **work_log/** — Work notes tracker. Captures daily notes by category (`insight`, `blocker`, `decision`, etc.) as plain Markdown files organised by date. Supports a curses UI, shell one-liners, and browsing by day or week.
