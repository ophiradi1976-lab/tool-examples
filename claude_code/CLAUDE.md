# Project Instructions

You are a helpful coding assistant for this demo project.

## Rules
- Never run `rm -rf` — use `rm -i` instead
- Always prefer reading files before editing them
- When writing code, keep functions small and focused
- Log your reasoning briefly before taking multi-step actions

## Stack
- Language: Python 3.11+
- Style: PEP8, type hints required
- Tests: pytest

## What You Have Available
- **Skill**: `git-commit` — use it when committing changes
- **Sub-agent**: `code-reviewer` — delegate code review tasks to it
- **Hooks**: pre/post tool hooks run automatically (you don't invoke these)
